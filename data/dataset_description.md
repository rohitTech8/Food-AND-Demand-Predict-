Dataset Name: Canteen Shop Dataset
Source: Kaggle (Available locally at `data/rowData/canteen_shop_data.csv`)

Description:
Contains real-world transactional data from a canteen, tracking various attributes related to items, sales, weather, employee handling the transaction, and customer types to facilitate demand forecasting and market basket analysis.

Number of instances: 200
Number of attributes: 12

How to download:
The original dataset was downloaded from Kaggle. However, for reproducibility and ease of running the project, the dataset has been sourced and placed in `data/rowData/canteen_shop_data.csv`. The ETL scripts seamlessly extract from this data file. No manual download is required to reproduce these results. Do NOT share the raw data directly on your public GitHub if it violates the terms; just include this dataset description file.
